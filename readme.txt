**This is an official plugin made together with [Persoo](http://persoo.co)**. Latest files available on [Github](https://github.com/vyskoczilova/persoo-for-woocommerce) and ready to fork.

**Enable personalized product recommendations, cross-sell and alternatives, super-fast personalized search and autocomplete to increase converions and revenue.**

Enjoy cloud-based **search and autocomplete** features with **filters, typo correction, custom results sorting, editable graphics and personalizations of brands, genders, price ranges** and more.

Simply add **carousels** with **alternative products** in the product page, or increase the average order value by adding smart **cross-sell products**, automatically calculated from products that are viewed or bought together. 

=== Requirements ===
* Account at [Persoo](http://persoo.co).
* Supports both WooCommerce 2.6.X & 3.1.X versions.
* The plugin uses "XML writer" PHP library for the product feed creation.


== Installation ==

1. Upload the plugin to your web site or install via plugin management.
1. Check whether the WooCommerce plugin is installed and active.
1. Activate the plugin through the `Plugins` menu in WordPress administration
1. Fill the necessary settings in `WooCommerce > Settings > Persoo`.
1. Done!
